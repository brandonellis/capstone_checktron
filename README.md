Checktron Desktop Application

npm install - Installs all required dependencies

npm build - Will generate a runnable .exe file for both OS's

To start the application in development mode:

npm start - runs the development server (Do this first)

npm run electron - will run the Checktron application

Please note that both 'npm start' and 'npm run electron' must be run in two separate command windows.
